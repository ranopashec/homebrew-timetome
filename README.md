# homebrew-timetome

Homebrew tap for [TimeTo](https://github.com/ranopashec/timetome2) — macOS menu bar time tracker.

## Install

```bash
brew tap ranopashec/timetome
brew install --cask timetome
```

Or in one line:

```bash
brew install --cask ranopashec/timetome/timetome
```

## Uninstall

```bash
brew uninstall --cask timetome
brew untap ranopashec/timetome
```
